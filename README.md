# FNP Sales Analysis Dashboard

## Project Overview
This project analyzes sales performance data for Ferns N Petals (FNP) to identify revenue trends, customer behavior, product performance, and operational insights. An interactive Excel dashboard is built to support data-driven business decisions.

## Business Objective
To provide management with a consolidated view of sales performance across time, occasions, products, and regions, enabling better planning, marketing optimization, and operational efficiency.

## Key KPIs
- Total Revenue
- Total Orders
- Average Order Delivery Time (Days)
- Average Customer Spend

## Dashboard Highlights
- Revenue analysis by occasion and category
- Monthly and hourly revenue trends
- Top-performing products
- City-wise order distribution

## Tools Used
- Microsoft Excel (Power Query, Pivot Tables, Charts)
- Data Cleaning & Transformation
- Dashboard Design & KPI Reporting
